BankInsight Data Migration

Import CSV data into the BankInsight PostgreSQL database using standardized templates.

Migration Process Flow

  1. Download and populate the CSV templates.
  2. Import data in order: customers, products, accounts, loans, then GL accounts.
  3. Review import summary and resolve row-level errors if present.
